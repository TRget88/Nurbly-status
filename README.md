# Nurbly Status

Live service status for [Nurbly](https://nurbly.com), served at
**https://status.nurbly.com** via GitHub Pages — independent of
Nurbly's own infrastructure, so this page stays reachable during an
outage.

Availability checks run live from the visitor's browser against
Nurbly's public health endpoints. Incidents and maintenance notices
are posted by editing [`incidents.json`](incidents.json) (schema
documented in the main repository's `ops/status-page/README.md`).
